# Intern Project

An implement of [iCheckPrint](https://www.figma.com/file/IBzeM7Qa0vsxUukNKuhEqW/iCheck-Print?node-id=30%3A2338&mode=dev)

## Requirements

- PHP 8.1+ (with Composer for package manager)
- NodeJS 18+

![](https://i.imgur.com/N8KMZTy.png)


## Build

```bash
composer install
npm install
npm run build
```

## Development

```bash
composer install
npm run dev-server
```
